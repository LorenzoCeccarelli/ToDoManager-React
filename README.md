# Installazione 

Installare node https://nodejs.org/en/download/package-manager/

  ```
  git clone https://github.com/LorenzoCeccarelli/ToDoManager-React.git
  cd toDoManager-React
  ```
  ## Server
  ```
  cd Backend
  npm install
  node server.js
  ```
  ## Client
  ```
  cd client 
  npm install
  npm start
  ```
